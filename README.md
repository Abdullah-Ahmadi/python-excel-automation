## The Problem
Manually updating and printing Excel documents every day is:

## The Solution
This Python-based automation handles the entire workflow for you:

- Uses your existing Excel template
- Automatically updates dates and values
- Prints only the required sheets
- Keeps a clear record of completed actions

No repetitive manual work. No missed entries.
